-- Spam Email Detection – Machine Learning Project --

This project implements a spam classification model using Python and Scikit-Learn.
It uses TF-IDF vectorization to convert text into numerical features and Naive Bayes to classify messages as Spam or Ham.

What the Code Does:-
Loads a sample dataset
Converts text into vectors
Trains a Naive Bayes model
Evaluates accuracy and performance
Allows you to test your own custom messages

How to Run:-
Install required modules:
pip install scikit-learn pandas

Run the Python file in PyCharm or any IDE.
